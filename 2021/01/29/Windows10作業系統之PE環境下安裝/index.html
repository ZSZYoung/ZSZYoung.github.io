    • Windows10作業系統之PE環境下安裝

    起因

    電腦在約兩週前進入bios時會花屏，控制變量後認為應該不是熒幕的問題，可能是bios或者硬體上的問題。然而但是重啟後還可以正常運作就沒在意，但之後發現電腦越來越卡頓，而且偶爾運作時會花屏。特別是打R6的時候那個每回合載入時間把對面和朋友都折磨的哇哇叫。其實本來也打算快要換電腦了（現在這個太習武之人，內存太小也不夠我多開虛擬機，整個硬體水平也跟不上了），但考慮到最近30系顯卡剛剛部署在移動端，目前的多為嘗鮮機比較貴，且產品線不太穩定，再過幾個月才應該是最佳的換機時間。所以我決定把電腦重裝下,希望再撐上幾個月。之前就用PE系統裝過Win系統，這次算是得心應手。

    事前準備

    • PE啟動U盤準備 個人比較推薦使用微PE製作PE啟動盤，東西乾淨而且內置很多有用工具。 http://www.wepe.com.cn/

      下載後打開選擇安裝PE系統到U盤即可，選項一般選擇默認的就行。

    • ISO檔準備 我們需要準備WIN10的光盤鏡像檔案（ISO檔）。 在大陸可以在下面網址獲得：https://msdn.itellyou.cn/

      在境外的話推薦直接從MS獲得:https://www.microsoft.com/zh-tw/software-download/windows10

      點擊立即下載工具然後打開，選擇創建安裝媒介，即可下載得到官方iso檔。

      注意是繁體作業系統環境請下載繁體版作業系統ISO，不然進入PE後安裝時選項會亂碼無法顯示。ISO檔可以存在U盤中，也可以放在機械硬盤/ssd中。

    重啟電腦進入bios

    在bios中調整啟動順序使得PE啟動盤啟動順序在最上邊。保存後reboot。

    在PE環境中操作安裝Win

    reboot後應該可以順利進入PE環境，在PE環境下的檔案管理器中找到原來準備好的ISO檔（請注意PE下的盤符和常規Win下命名可能不一致）。雙擊掛載後打開Setup即可開始安裝步驟。按照提示選擇需要的選項即可，之後等待安裝完即完成新系統安裝。

    一些可能遇到的問題

    • 如果要安裝新系統的機械硬盤/ssd空間不足需要格式化時，要連帶把原本系統分隔出的分卷格式化。否則會報錯無法完成自動的分卷。
    • 進入全新系統後發現動畫不流暢不要擔心，可能是核顯(有核顯下)的驅動過於老舊造成。簡單點可以下載驅動精靈更新下核顯驅動即可獲得流暢動畫體驗。
    • 盡量選擇在PE下進行新系統安裝。儘管現在可以在本系統中直接用ISO檔進行重裝系統，但是PE下重裝成功率更高且可以應對無法boot的情況。
    • error code: 0x8007025D. 改為使用MS官方media creation tool後解決，考慮可能是ISO檔多次拷貝的缺失性問題。
    • Windows cannot find the Microsoft license software terms.改為使用MS官方media creation tool後解決，考慮可能是ISO檔多次拷貝的缺失性問題。

    獲得重裝介質的其他備選項

    • 用rufus配合iso檔製作可boot的USB flash
    • 使用MS官方的media creation tool將USB flash製作為可boot安裝媒介

    後記

    更新完系統後確實流暢度得到了很大的改善，但還是出現偶爾花屏且不如原來流暢的情況，考慮應該是確實有硬體上的問題。希望可以再撐幾個月，在比較好的時間點換電腦。 之前聽說東別重裝個系統都要收好像400左右，清個灰要1000多，真的是嚇死了。這不是有手就行@@，也許我應該開個A同學電腦坊，物美價廉得競爭一下。 羅列下成本：

    • 重裝系統：U盤(可反復利用...)
    • 清灰: 螺絲器具套裝,硅脂(買好一點一管也就100多點，可以用個好多次)，氣吹等除塵用品，嚴謹點買個防靜電手套、嚴謹點買個放電手環。

    暴利啊，我還學什麼CS，保養電腦都起飛了@@。
